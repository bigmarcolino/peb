Sempre instalar a versão mais recente dos programas

01- instale o XAMPP (não altere o diretório padrão de instalação) -> https://www.apachefriends.org/download.html

02- instale o Sublime Text -> https://www.sublimetext.com/3

03- instale o Composer -> https://getcomposer.org/Composer-Setup.exe

04- instale o Git -> https://git-scm.com/downloads

05- instale o Node -> https://nodejs.org/en/

06- instale o Windows Credentials Manager (arquivo .exe) -> https://github.com/Microsoft/Git-Credential-Manager-for-Windows/releases/tag/v1.10.1

07- crie uma conta no GitLab -> https://gitlab.com/users/sign_in
	
08- passe para mim (Marcus) o email da conta criada no GitLab, para que eu possa autorizar o download do projeto

09- com o Sublime Text, abra o arquivo httpd-vhosts.conf, que está na pasta C:\xampp\apache\conf\extra

10- no final do arquivo, adicione o código a seguir:

	# VirtualHost for LARAVEL.DEV

	<VirtualHost laravel.dev:80>
		DocumentRoot "C:\xampp\htdocs\projeto-escoliose-brasil\public"
  		ServerAdmin laravel.dev
  		<Directory "C:\xampp\htdocs\projeto-escoliose-brasil">
    		Options Indexes FollowSymLinks
    		AllowOverride All
    		Require all granted
  		</Directory>
	</VirtualHost>
		
11- copie o arquivo hosts, que está na pasta C:\Windows\System32\drivers\etc, e cole na área de trabalho

12- com o Sublime Text, abra o arquivo hosts da área de trabalho

13- no final do arquivo, adicione o código a seguir:

	127.0.0.1	laravel.dev

14- copie arquivo hosts da área de trabalho para a pasta C:\Windows\System32\drivers\etc e confirme a substituição do arquivo

15- abra o XAMPP Control Panel, clique no botão config com um ícone de uma ferramenta, marque Apache e MySQL e clique em Save

16- na linha do Apache e na linha do MySQL, clique em Start

17- clique em Admin, na linha do MySQL. O navegador será aberto

18- no navegador, do lado esquerdo, clique em New

19- no campo Create database, digite projeto-escoliose-brasil. No campo Collation, escolha utf8mb4_unicode_ci (é a penúltima opção) e clique em Create

20- feche o navegador

21- abra o prompt de comando

22- copie e cole no prompt de comando: cd\ && cd xampp/htdocs && git clone https://mvinicius@gitlab.com/mvinicius/projeto-escoliose-brasil.git && cd projeto-escoliose-brasil && composer install && npm install && npm install -g bower && bower install && npm run dev && php artisan key:generate && php artisan migrate, tecle Enter e aguarde o processo terminar

23- se seu email e senha do GitLab forem requisitados, basta digitá-los

24- feche o prompt de comando

25- abra http://laravel.dev no navegador para visualizar o sistema


Toda vez que precisar testar o sistema, abra o prompt de comando,
digite cd\ && cd xampp/htdocs/projeto-escoliose-brasil e tecle Enter, 
digite git pull e tecle Enter (este comando atualizará o sistema para a versão mais atual).
Abra o XAMPP Control Panel, depois abra o navegador e digite http://laravel.dev
